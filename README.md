## About the Project

**El Impostor Backend** is A lightweight and fast backend for ElImpostor, built with Express.js, providing RESTful APIs, real-time data handling, and seamless integration with the frontend.

## Built With

This project was developed using:

- **JavaScript** for client-side functionality
- **Node.js** for backend development
- **Express** for web application framework

Repository: [github.com/JMF-Alex/ElImpostor-Backend](https://github.com/JMF-Alex/ElImpostor-Backend)

## Getting Started

### Prerequisites

No external dependencies required.
Only a **web browser** (Chrome, Edge, Firefox, etc.).

### Installation

1. Clone the repository:

```bash
git clone https://github.com/JMF-Alex/ElImpostor-Backend.git
```

2. Enter the project directory:

```bash
cd ElImpostor-Backend
```

3. Open `index.html` in your browser.

That's it â€” the app runs entirely locally!

## Usage

Once opened, you can:

- Navigate through the application interface
- Use the main features as intended
- Data remains saved locally in your browser
